# CAD редактор блока

## Назначение

`BlockCadEditorModal` предназначен для ручного построения и корректировки контура блока внутри контура здания в локальной метрической системе координат.

## Ключевые возможности

- Автоповорот здания в рабочем поле: длинная сторона здания выравнивается по горизонтали.
- Редактирование узлов блока: добавление точек кликом и перетаскивание существующих вершин.
- Привязка (snap):
  - к сетке (1 метр),
  - к вершинам здания,
  - к сторонам здания (проекция точки на ребро).
- Переключаемые режимы привязки в UI (grid / vertex / edge).
- Визуальные метрики:
  - длины сторон здания и блока в метрах (`xx.xx м`),
  - площадь здания и блока внутри контуров (`xx.xx м²`),
  - индикация последнего типа привязки и её расстояния.
- Валидация перед сохранением:
  - минимум 3 точки,
  - отсутствие самопересечений контура блока,
  - минимальная длина стороны (`0.30 м`),
  - блок полностью внутри контура здания.

## Новый режим: геометрия по размерам

- Добавлен инструмент построения прямоугольного контура по параметрам:
  - ширина (м),
  - глубина (м),
  - ручной поворот на экране (ползунок градусов).
- Опорной точкой (anchor) служит:
  - выбранная вершина блока,
  - иначе первая вершина текущего контура,
  - иначе первая вершина здания.
- Опция «Прижать вершины шаблона к границам здания» выполняет привязку каждой вершины шаблона к ближайшей стороне здания.
- Перед применением и сохранением шаблон проходит те же проверки геометрии.

## UX-фичи

- Undo/Redo:
  - кнопки в интерфейсе,
  - горячие клавиши `Ctrl+Z` и `Ctrl+Shift+Z`.
- Выбор вершины кликом и удаление выделенной вершины:
  - кнопка «Удалить выбранную вершину»,
  - горячие клавиши `Delete` / `Backspace`.
- `Esc` сбрасывает выделение и active drag.
- Визуальный маркер активной привязки:
  - жёлтый — привязка к вершине,
  - розовый — привязка к стороне,
  - светлый — привязка к сетке.
- Предпросмотр геометрии по размерам отображается пунктиром в рабочей области.

## Ограничения

- Импорт/экспорт форматов намеренно не реализованы на текущем этапе.
- Порог привязки фиксирован (`1.2 м`) для привязки к вершинам/сторонам здания.

## Техническая реализация

Основной файл реализации:

- `src/components/cad/BlockCadEditorModal.jsx`

Основные группы логики:

1. **Геометрические утилиты внутри компонента**: расстояния, проекция на сегмент, проверка самопересечений, контроль минимальной длины ребра.
2. **Подготовка сцены**: перевод геометрии здания/блока в локальные метры, поворот и масштабирование в `svg` viewport.
3. **Редактор состояния**: draft-точки, режимы привязки, выделение вершины, undo/redo стек, индикатор последнего snap.
4. **Геометрия по размерам**: генерация прямоугольника от anchor с поворотом и опциональным прижатием к границам здания.
5. **Валидация и сохранение**: обратный поворот, перевод в географические координаты и проверка вложенности.
